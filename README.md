# Space Adventure Quiz

About the Project

This project is a very beginner-friendly python project to test your knowledge on space. The user has to answer 5 questions regarding planets, Moon, Sun and solar system. The program checks if the answer is correct, stores the score and displays the final result.

Features

* 5 Space related questions

* Automatically checking the answers

* Scoring the points

* Greeting message with player name
* Displaying final message based on the score

* Running it on terminal

What I Learned

* How to display text using print()
* How to take user input using input()
* How to store data in variables
* How to use if, elif, else
* How to add, update and display score

* How to compare text

How to Run the Project

1. Download spaceadventurequiz.py.

2. Ensure that you have installed python in your system.

3. Open the file on a python IDE or open the file in your terminal and type

``bash

python spaceadventurequiz.py

``

4. Answer the questions and see your final score.

Requirements

* Python 3

* No extra libraries needed
